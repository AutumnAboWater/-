# Java☕
核心技术栈：Java+JDBC+Maven+JavaSwing+MySQL+JDBC+百度千帆
## 人事后台管理系统💛
### 使用maven构建MySQL及JDBC😀
### 创建MySQL数据库及表插入模拟数据😶
### 主要功能模块：登录、注册、人事员工信息CRUD 特殊功能：AIchat（引入百度文心一言API实现AI对话）🏝

# 部署本地/快速开始/安装起步
* 1.下载或git clone项目
* 2.运行数据库文件./SQL/XX企业人事后台管理系统.sql
* 3.修改数据库配置文件./util/DButil.java
* 4.修改BaiDu文件夹中的API配置(需要自行去百度千帆官网获取)
* 5.运行Main.java
